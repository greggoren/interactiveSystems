Welcome!
Thanks a lot for participating in this experiment.
In this experiment we simulate a search engine environment. The experiment includes the following stages:

  1. Choose a query of interest
  2. After submitting the query you will be presented with 5 documents - the documents are ordered according to their relevance scores assigned by the search engine's algorithm (from most to least relevant)
  3. Re-enter the query. You will be presented with a list of modified documents ordered via the same logic in previous step. This stage, simulates a re-ranking procedure done by the search engine. In real systems, documents are constantly changing. Hence, once in a fixed period, all documents are re-indexed by the search engine and are re-ranked with respect to different queries
  4. Fill a summarizing form

The algorithm which is the basis of the search engine considers the following three factors:
  1. Quality features - signals that portray the quality of the document in terms of readability and vocabulary
  2. Language modelling features - signals that portray how query terms are distributed within the content of the document
  3. Term frequency features - signals that portray how frequent are the query terms within the content of the document
